ContactCard(props) → {JSX.Element}

Componente ContactCard que muestra los detalles de contacto de un profesional, incluyendo nombre, teléfono, correo electrónico, dirección, género y calificación. También proporciona botones para editar y eliminar el contacto.
Parameters:

props Object Propiedades del componente.
Properties
Name Type Description
practitioner Object Datos del profesional a mostrar.
Properties
Name Type Description
id string Identificador único del profesional.
name Array.<Object> Array que contiene los detalles del nombre (primer y apellido).
telecom Array.<Object> Array que contiene los datos de contacto, como teléfono y correo electrónico.
address Array.<Object> Array con la dirección del profesional (línea de dirección, ciudad, código postal, país).
gender string Género del profesional.
qualification Array.<Object> Array que contiene la calificación profesional, con detalles sobre el cargo.
onDelete function Función para eliminar el contacto, pasando el ID del profesional.
onEdit function Función para editar el contacto, pasando el objeto completo del profesional.

ProtocoloView()

Vista para mostrar las patologías y ubicaciones asociadas a una especialidad médica. Esta vista permite al usuario seleccionar una especialidad médica y, en función de ella, se cargan las patologías relacionadas. Al seleccionar una patología, se muestra información adicional sobre la misma, como observaciones, criterios y exámenes. También se muestran las ubicaciones asociadas a los servicios de salud relacionados con la especialidad y patología seleccionadas en un mapa interactivo.
